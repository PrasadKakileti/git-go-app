# Frontend - JobHub UI

Modern, responsive user interface for the Job Portal application.

## Structure

```
frontend/
├── index.html          # Main landing page
├── css/
│   └── style.css      # Styles with animations
├── js/
│   └── app.js         # Interactive features
└── images/            # Image assets
```

## Features

### User Experience
- **Smooth Scrolling**: Navigate seamlessly between sections
- **Animated Counters**: Eye-catching statistics
- **Form Validation**: Real-time feedback
- **Success Animations**: Confetti effect on registration
- **Loading States**: Visual feedback during API calls
- **Responsive Design**: Works on all devices

### Design Elements
- Modern gradient backgrounds
- Card-based layout
- Hover animations
- Sticky navigation
- Mobile-first approach

### Sections
1. **Hero**: Eye-catching intro with CTA
2. **Features**: 6 key benefits
3. **Registration**: User-friendly form with:
   - Email input
   - City dropdown (10 major cities)
   - Daily/Weekly frequency selector
   - Instant feedback

## Color Scheme
- Primary: #6366f1 (Indigo)
- Secondary: #8b5cf6 (Purple)
- Success: #10b981 (Green)
- Error: #ef4444 (Red)
